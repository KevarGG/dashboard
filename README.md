# University Student Analytics Dashboard
# Estudiantes: Kevin Arrieta, Juan Niño, Anderson Ortega, Enrique Hernandez, Ricardo Henriquez

Dashboard interactivo construido con Streamlit para analizar los datos de admisión, retención y satisfacción de estudiantes.

## Archivos
 app.py = código del dashboard
 university_student_data_clean.csv = dataset limpio
 requirements.txt = dependencias necesarias

# Cómo ejecutarlo localmente
1. Instalar dependencias: `pip install -r requirements.txt`
2. Ejecutar la app: `streamlit run app.py`
